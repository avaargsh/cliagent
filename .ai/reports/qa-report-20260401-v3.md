## 3. 本轮新发现缺陷

### QA-005：CLI 全局 flag 位置与规范示例不一致

- 级别：P1
- 现象：
  - `PYTHONPATH=src python3 -m digital_employee.api.cli.main config show --json`
  - 返回 `exit 2`
  - stderr：`unrecognized arguments: --json`
- 期望：
  - `.ai/temp/cli-spec.md` 的示例默认允许把 `--json` 放在子命令后。
- 实际：
  - 当前实现基于标准 `argparse`，只支持 `dectl --json config show`。
- 影响：
  - 属于 CLI 体验/文档偏差，不影响本轮 OpenAI provider 的运行时修复和自动化回归结果。
- 建议：
  - 二选一：
    - 修正文档示例，统一要求全局 flag 放在子命令前；
    - 或在 CLI 层做兼容解析，接受子命令后的全局 flag。
